The Minimizer structure provides functionality for minimizing values. A minimal value is semantically equivalent but is represented internally using a maximal amount of sharing. E.g., the values (SOME 4, SOME 4) and let val a = SOME 4 in (a,a) end are structurally equivalent (and hence considered equal by Alice), but their representations are not. Minimization would transform the internal representation of the first value into that of the second.
