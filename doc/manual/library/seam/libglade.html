    signature GLADE
    structure Glade : GLADE

The Glade structure provides access to Libglade. This library allows a program to load its user interface from an XML description at runtime. The XML file format is that of the user interface builder GLADE, which allows you to create the user interface interactively.

See the overview page for a general introduction to the GTK library binding.

See also: GLib, Pango, Atk, Gdk, Gtk, Canvas 

    import signature GLADE from "x-alice:/lib/gtk/GLADE-sig"
    import structure Glade from "x-alice:/lib/gtk/Glade"
  
signature GLADE =
sig
    val load :  { domain : string option, filename : string, root : string option } -> Gtk.object
    val connect : Gtk.object * (string * Gtk.callback_function) list -> unit
    val getWidget : Gtk.object * string -> Gtk.object
end
load { domain, filename, root }

Creates a new GladeXML object (and the corresponding widgets) from the XML file filename. Optionally it will only build the interface from the widget node root (iff it is not NONE). This feature is useful if you only want to build say a toolbar or menu from the XML file, but not the window it is embedded in. You can also specify a different translation domain if you do not wish to use the default one.

connect (gladexml, callbacks)

In the glade interface descriptions signal handlers are specified by name. This function allows you to connect alice functions to the signals.

getWidget (gladexml, widgetName)

In the glade interface descriptions every widget is given a name. This function allows you to get the widget with name widgetName.
